Documentation for Lychee
========================

*Lychee* is the MEI document management system developed by `nCoda <https://ncodamusic.org/>`_.

If you do not know what *Lychee* is, please start with the :ref:`basic` section.

.. important::
    *Lychee* is prototype software. We welcome you to use *Lychee* as you see fit, but please be
    aware that we intend to change the *Lychee* interface considerably in the near future.

    For more information on what this means, please refer to the :ref:`basic-introduction` section.
