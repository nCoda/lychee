Namespace Constants for XML
===========================

String constants to ensure you use tag and attributes names in the proper XML namespace.
For the :ref:`namespace-mei`, :ref:`namespace-xlink`, and :ref:`namespace-xml`.

Tag and attribute names will never be removed or changed unless removed or changed in the
corresponding specification. Names are usually added when they are needed within *Lychee*, so there
are many names in the specifications missing from this module. Please feel free to add them.

If you wish to use these modules outside *Lychee*, I really wish you had better things to do with
your life than think about XML namespaces.
